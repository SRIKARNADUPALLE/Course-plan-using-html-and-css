    • Course Plan

    Course Title: 24CSE441 Full stack development with java

    Instructor: Nadupalle Srikar

    Duration: 14 Weeks

    Objective: To provide students with a solid foundation in full stack web development with java and spring framework in both front end and backend technologies.

    Course Outline

    • Week 1:Introduction to Java and Object-Oriented Programming
    • Week 2:Strings and Files in Java
    • Week 3:Collections Framework in Java
    • Week 4:Review of HTML, CSS, and JavaScript
    • Week 5:Front-End Development with Angular (Part 1)
    • Week 6:Front-End Development with Angular (Part 2)
    • Week 7:Angular Pipes and Forms
    • Week 8:Review of Data Structures in Java
    • Week 9:Back-End Development with Spring Boot (Part 1 and Part 2)
    • Week 10:RESTful Web Services and Spring Boot Security (Part 1 and Part 2)
    • Week 11:Testing and Deployment
    • Week 12:Integrating Angular with Spring Boot
    • Week 13: Building a Simple Web Project
    • Week 14: Project Presentation and Evaluation

    Assignments, quizes, projects

    Students will be given weekly assignments and will be required to complete a final project that showcases the skills learned throughout the course.

    Evaluation Policy

    • Mid term: 20%
    • Continuous Evaluation(Quiz (5 quizzes weightage 4)- 20 Marks & Project (Frontend-15 Marks + Backend 15Marks)- 30Marks): 50%
    • End semester: 30%
    • Total: 100 Marks
